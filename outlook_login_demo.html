⚙️ Cấu hình EmailJS

Điền thông tin EmailJS để gửi OTP thật qua email.

Hướng dẫn lấy thông số (miễn phí):
1. Vào emailjs.com → Đăng ký tài khoản
2. Email Services → Add Service → Gmail/Outlook → copy Service ID
3. Email Templates → Create Template → nội dung dùng {{otp_code}} và {{to_email}} → copy Template ID
4. Account → General → copy Public Key
Vui lòng điền đầy đủ cả 3 thông số.
